📐 LaTeX → Moodle XML (полный шаблон)

Точное соответствие формату: multichoice, стиль times new roman, CDATA + обёртка <p><span>...</span></p>, \correctchoice → fraction="100"

📥 Исходные вопросы (LaTeX)

✅ Ожидание ввода. Нажмите «Конвертировать».
📌 Требования к формату вопроса:
• Каждый вопрос начинается с \question (текст вопроса).
• Затем \begin{randomizechoices} ... \end{randomizechoices}.
• Варианты: \choice (неверный) и \correctchoice (верный, 100%).
• Поддерживается LaTeX-разметка: $\sigma$ → преобразуется в $$\sigma$$ для совместимости с шаблоном.
• Результирующий XML полностью повторяет структуру из вашего шаблона: каждый ответ обёрнут в <p><span style="font-family: 'times new roman', times, serif; font-size: medium;">...</span></p>.
